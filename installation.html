how to install python

Back to Home

for windows users

navigate to python.org

click on the downloads option in the menu bar

now click on the download latest versions

open exe

check things shown in pic

click customise install

now add the extras you want to be installed now hit next

now hit install

now hit close button

for mac users

navigate to python.org

click on the downloads option in the menu bar

now click on the download latest versions

open the pkg file

click continue on the introduction screen

click continue on the read me screen

click continue on the license screen then click agree

select your destination disk and click continue

click install and enter your password if prompted

wait for installation to complete then click close

how to install idle on windows

idle comes with python by default

search for idle in start menu

click on idle python to open it

if idle doesnt appear reinstall python with tcl tk option checked

how to install idle on mac

idle comes with python by default

open finder and go to applications

find python folder and open it

double click on idle to open it

alternatively open terminal and type idle3

idle3

test python installation

for mac users run this command

python3 --version

for windows users run this command

python --version
